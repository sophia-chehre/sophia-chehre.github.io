Project 01 · Transfer Learning

Building Detection with Transfer Learning

This project focused on improving building footprint extraction by adapting a pretrained deep learning model in ArcGIS Pro. Instead of relying on inference only, the workflow used custom training samples, model fine-tuning, and before/after comparison to show how transfer learning improved extraction quality under new imagery conditions.

Workflow structure

The workflow below highlights the main technical stages rather than retelling the tutorial step by step.

Stage 01

Input review and image preparation

The project started with reviewing the source imagery and preparing the RGB representation needed for sample creation, interpretation, and model adaptation.

Stage 02

Custom sample creation

Training samples were created to represent the local building patterns and provide the model with examples from the new image context.

Stage 03

Transfer learning and model training

The pretrained model was fine-tuned in ArcGIS Pro so that extraction behavior could be adjusted to the target imagery rather than remaining generic.

Stage 04

Before/after comparison

Final evaluation focused on comparing the baseline extraction result with the refined output after transfer learning, making the effect of model adaptation visible.

Transfer learning in this project was used to refine an existing model for a new dataset, not to build a deep learning workflow entirely from scratch.
