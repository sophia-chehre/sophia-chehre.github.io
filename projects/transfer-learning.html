Project 03

Building Extraction with Transfer Learning

A pretrained deep learning model was fine-tuned with custom training samples in ArcGIS Pro to improve building footprint extraction results and adapt the model to new imagery conditions.

View Web App

Project Focus

This project demonstrates a transfer learning workflow for improving building detection in ArcGIS Pro. The goal was to move beyond a generic pretrained model and refine it with new training data so that extraction quality better matched the target imagery and local building patterns.

Software Skills Demonstrated

Deep Learning Workflow

  • Worked with a pretrained building extraction model
  • Prepared custom training data for model improvement
  • Compared baseline and refined detection outputs

Training Data Preparation

  • Reviewed input imagery and RGB extraction
  • Created training samples for transfer learning
  • Prepared data for model fine-tuning in ArcGIS Pro

Model Evaluation

  • Assessed detection results before transfer learning
  • Tracked model training progress
  • Compared improvements after fine-tuning

Web-Based Presentation

  • Published results for interactive comparison
  • Used a web app to communicate model improvement
  • Structured outputs for portfolio presentation

Workflow Summary

1. Input Review

The workflow started with reviewing the source imagery and preparing the RGB input needed for building extraction and training sample creation.

2. Sample Preparation

Custom training samples were created to represent the building patterns in the target dataset and provide new examples for model refinement.

3. Transfer Learning

The pretrained model was fine-tuned using the new training data in ArcGIS Pro to improve detection performance on the selected imagery.

4. Result Comparison

Building extraction results were compared before and after transfer learning to evaluate how model adaptation improved the final output.

Project Outcome

This project shows how transfer learning can improve building extraction without building a deep learning workflow entirely from the beginning. It demonstrates model adaptation, training sample preparation, comparative evaluation, and clear presentation of results in a portfolio-ready format.
