Project 04

Extract High-Resolution Land Cover with GeoAI

GeoAI-based land cover classification, vector refinement, and green space extraction using high-resolution imagery in ArcGIS Pro and ArcGIS Online.

View Web App

Project Focus

This project demonstrates a complete GeoAI workflow for high-resolution land cover mapping, starting from orthomosaic imagery and moving through classification, refinement, thematic filtering, and interactive web visualization.

The goal was not only to produce a classified land cover result, but also to turn the raw output into cleaner GIS-ready layers that are easier to interpret, compare, and present in a portfolio setting.

Input Imagery

The workflow started with a high-resolution orthomosaic used as the input imagery for classification. This image provided the visual detail needed to distinguish urban surfaces, vegetation, and water-related areas across the study site.

High-resolution orthomosaic used as the base input for GeoAI land cover classification.

Initial Land Cover Classification

The first output of the workflow was a raster-based land cover classification. This result contained multiple classes including water, wetlands, tree canopy, shrubland, low vegetation, barren surfaces, structures, impervious surfaces, and impervious roads.

This stage represents the direct classified model output and provides the full thematic structure of the site before further refinement.

Initial raster output showing the classified land cover result across the study area.

Refined Land Cover Features

After classification, the raster result was converted into vector features and symbolized as a cleaner land cover layer. This step made the output easier to manage, interpret, and reuse in a more controlled GIS workflow.

Compared with the raw raster output, this refined feature layer provides a more presentation-ready version of the classification and better reflects GIS post-processing skills beyond the model itself.

Vectorized and refined land cover layer derived from the classified raster result.

Green Space Extraction

A separate green space layer was derived by isolating vegetation-related classes from the land cover result. In this project, the extracted green spaces mainly focused on low vegetation and tree canopy.

This step adds analytical value to the workflow by moving from a full classification result to a more focused thematic output. It shows how classification can support interpretation and targeted spatial analysis.

Green spaces were extracted from the classified land cover result by filtering vegetation-related classes into a separate thematic layer.
Extracted green space layer overlaid on the imagery for focused interpretation of vegetation patterns.

Interactive Web Visualization

The final outputs were published to ArcGIS Online and configured in an interactive web application. This allowed the classified layers and the extracted green spaces to be explored in a clean presentation format, while also making it possible to compare them with the original imagery.

The project therefore demonstrates not only GeoAI classification, but also the full workflow from image-based analysis to interactive web delivery.

Project Outcome

This project shows a complete workflow that combines image-based classification, raster-to-vector refinement, thematic filtering, and web-based communication. The result is not only a classified map, but a structured GIS output that can support interpretation, comparison, and presentation.
