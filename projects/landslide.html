Software skills demonstrated

GeoAI and deep learning inference

  • Used a pretrained model from ArcGIS Living Atlas to detect building footprints from aerial imagery.
  • Worked with object detection output as a feature layer for later spatial comparison.
  • Integrated AI-based extraction into a broader GIS workflow rather than stopping at detection.

Raster-based susceptibility analysis

  • Applied raster analysis in ArcGIS Online to classify landslide susceptibility.
  • Worked with multiple environmental input layers to generate the risk surface.
  • Interpreted results at island scale before moving to local overlay review.

Overlay and exposure mapping

  • Compared the building extraction layer with susceptibility classes to identify exposed structures.
  • Used spatial overlay logic to move from separate outputs to decision-relevant results.
  • Worked across both broad and zoomed-in scales of analysis.

Web-based communication

  • Published outputs through an ArcGIS Online web map and Instant App.
  • Configured interactive review through map layers, context, and app presentation.
  • Connected technical analysis to a presentation format that is easier to assess quickly.

Workflow structure

The workflow below highlights the main technical stages instead of retelling the tutorial line by line. That makes the logic easier to review quickly.

Stage 01

Building extraction from imagery

Aerial imagery was processed in ArcGIS Online using a pretrained deep learning model to generate building footprints for the study area.

Stage 02

Susceptibility surface creation

Landslide susceptibility was modeled through raster analysis using elevation, soil types, land use, and distance to rivers as input layers.

Stage 03

Overlay-based exposure analysis

The building layer was compared with the raster risk surface to identify structures located in moderate to very high susceptibility zones.

Stage 04

Interactive delivery

The final outputs were published as an interactive web map and Instant App so the results could be explored beyond the analysis environment.

Input layers used for susceptibility modeling included elevation, soil types, land use, and distance to rivers. This is important because the raster result was not a standalone visualization but a multi-input analytical surface.
