Palm Tree Detection with a Pretrained Model

This project demonstrates a simple pretrained-model workflow in ArcGIS Pro. A palm tree detection model was applied to high-resolution aerial imagery to generate bounding box detections and produce a structured feature output for visual review.

Project overview

Compared with the other projects in this portfolio, this one is more straightforward. Its value lies in showing a clear GeoAI deployment workflow: selecting a suitable pretrained model, running detection on aerial imagery, and reviewing the resulting output as a GIS layer.

What this project shows

Pretrained model deployment

The project shows how a ready-to-use deep learning model can be applied in ArcGIS Pro without retraining or model adaptation.

Main result

Structured palm detections

The output is a detection layer of palm trees represented as bounding boxes across the study area.

Software skills demonstrated

This project is simpler than the others, so the page focuses only on the core technical skills that are actually visible in the workflow.

Model application

  • Selected a pretrained palm tree detection model.
  • Applied the model to high-resolution aerial imagery in ArcGIS Pro.
  • Ran an inference-based object detection workflow.

Output review

  • Generated palm tree detections as bounding box features.
  • Reviewed the result across the study area and in close-up view.
  • Used the output as a structured GIS-ready detection layer.

Workflow structure

The workflow is short and direct, which matches the nature of the project.

Stage 01

Input imagery

High-resolution RGB aerial imagery of the study area was used as the input for palm tree detection.

Stage 02

Pretrained detection

A palm tree detection model from ArcGIS Living Atlas was applied in ArcGIS Pro to generate object detections.

Stage 03

Output generation

The model produced a detection layer of palm trees represented as bounding boxes.

Stage 04

Visual review

The result was inspected both across the full study area and at close range to review the detected objects.
