Detect Objects with a Deep Learning Pretrained Model

Automated palm tree detection in high-resolution aerial imagery using a pretrained deep learning model in ArcGIS Pro.

Overview

This project demonstrates object detection using a pretrained deep learning model in ArcGIS Pro. The goal was to automatically detect palm trees in high-resolution aerial imagery and generate a feature layer representing the detected objects.

Study Area & Data

The workflow used high-resolution RGB aerial imagery of Kolovai. The imagery contains vegetation patterns, agricultural plots, and clusters of palm trees, making it suitable for pretrained object detection.

Detection Workflow

  • Loaded Kolovai imagery in ArcGIS Pro.
  • Applied a pretrained Palm Tree Detection model from ArcGIS Living Atlas.
  • Executed the Detect Objects Using Deep Learning tool.
  • Generated bounding boxes representing detected palm trees.
  • Exported the detections as a feature class for further GIS analysis.

Detection Settings

The detection was run with the pretrained model using the default workflow in ArcGIS Pro, including confidence-based filtering and Non Maximum Suppression (NMS) to reduce overlapping detections. During troubleshooting, system decimal formatting had to be corrected to ensure the model parameters were interpreted properly.

Tools Used

  • ArcGIS Pro
  • ArcGIS Living Atlas
  • Detect Objects Using Deep Learning
  • Pretrained object detection model
  • Raster analysis

Results

The model successfully detected a large number of palm trees across the study area. The resulting output demonstrates how pretrained deep learning models can be integrated into GIS workflows for automated feature extraction from imagery.
