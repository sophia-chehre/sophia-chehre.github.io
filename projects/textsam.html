Detect Objects with TextSAM

Prompt-based object detection in high-resolution aerial imagery using the TextSAM model in ArcGIS Pro.

Overview

This project explores prompt-based object detection in high-resolution aerial imagery using the TextSAM model in ArcGIS Pro. The objective was to automatically detect boats in a marina environment and convert the detections into vector features that can be used in GIS workflows.

Study Area & Data

High-resolution aerial imagery of a coastal harbor area was used as the input dataset. The imagery contains multiple boats located along docks and marina structures, allowing individual vessels to be distinguished from surrounding water surfaces.

Detection Workflow

  • Prepared and loaded the aerial imagery in ArcGIS Pro.
  • Ran TextSAM inference using a text prompt describing the target object.
  • Generated candidate detection polygons from the model output.
  • Converted the detections into vector features for further analysis.

Detection Strategy

To improve detection accuracy, the output was refined after the initial inference step. Detections were filtered based on confidence levels, and very small geometries were removed to reduce false positives. This post-processing step helped produce a cleaner and more reliable output layer for GIS use.

Tools Used

  • ArcGIS Pro
  • TextSAM
  • Deep Learning tools (ArcGIS)
  • Image analysis
  • Vector feature extraction

Results

The final detection layer successfully identified most boats located within the marina area. The comparison between the initial detection and the refined output highlights the importance of confidence filtering and post-processing in GeoAI object detection workflows.
