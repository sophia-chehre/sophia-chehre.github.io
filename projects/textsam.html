Detect Objects with TextSAM

Object detection in high-resolution aerial imagery using the TextSAM model in ArcGIS Pro.

Overview

This project demonstrates how a prompt-based GeoAI model can be used to detect objects in aerial imagery. The workflow was completed in ArcGIS Pro using the TextSAM model.

Workflow

  • Prepared the imagery in ArcGIS Pro.
  • Applied the TextSAM model for object detection.
  • Generated initial detection results.
  • Refined the output through post-processing.

Tools Used

  • ArcGIS Pro
  • TextSAM
  • Deep Learning tools
  • Image analysis
  • Vector feature extraction
